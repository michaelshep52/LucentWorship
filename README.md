🚀 LucentWorship

Modern worship presentation software built for simplicity, speed, and smaller teams.

LucentWorship is a lightweight, cross-platform presentation system designed to help churches display lyrics, scripture, and media seamlessly—without the complexity or high cost of traditional solutions.

🚀 Getting Started
git clone https://github.com/michaelshep52/LucentWorship.git
cd LucentWorship
Install dependencies: `npm install`
Create an `.env.local` file and set the right environment variables

VITE_SUPABASE_URL=your_app_url
VITE_SUPABASE_ANON_KEY=your_anon_key (Project settings -> API Keys -> Legacy anon)

Run the app: `npm run dev`
